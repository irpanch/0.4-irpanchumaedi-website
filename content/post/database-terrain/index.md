**DEMNAS** adalah Digital Elevation Model Nasional Indonesia resolusi tinggi yang diproduksi oleh
**Badan Informasi Geospasial (BIG)** menggunakan data radar IFSAR (resolusi 8.3m) dan
TerraSAR-X (resolusi 27m). Data dapat diunduh **gratis** langsung dari server BIG.

## Cara Mengunduh

1. Klik **Lihat Daftar Tile** pada wilayah yang dibutuhkan
2. Cari NLP / kode lembar peta yang diperlukan
3. Klik tombol **⬇ Unduh** — file diunduh langsung dari server BIG

> 💡 **Tips:** Gunakan **Ctrl+F** di halaman daftar tile untuk mencari kode NLP spesifik.
